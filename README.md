# DCIT 308 Individual Assignment Starter Code

Compile and run from the `src` folder:

```bash
javac *.java
java GhanaServiceSimulator ../data/requests_sample.csv
```

Replace `requests_sample.csv` with your own Ghana-context dataset before submission.

Important: The starter code contains TODO markers. Each TODO is explained in the assignment brief. You must implement the missing logic and be able to explain it during tutorial/oral defence.
